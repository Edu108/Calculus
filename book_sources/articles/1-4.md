###### Inverse Trigonometric Functions

        經過前述討論，我們知道重新給定定義域後，可以是非一對一的函數有反函數。因此，我們想要求得三角函數的反函數，使該函數能透過三角比給出夾角。

### 反三角函數的值域

        如圖所示，如果我們重新設定三角函數定義域，我們就可以求出相對應的反三角函數。各函數的定義域如下表所列:

| 函數  | 定義域                            | 值域     |
| --- | ------------------------------ | ------ |
| sin | [-\frac{\pi}{2},\frac{\pi}{2}] | [-1,1] |
| cos | [0,\pi]                        | [-1,1] |
| tan | (-\frac{\pi}{2},\frac{\pi}{2}) |        |
| cot | (0,\pi)                        |        |
| sec |                                |        |
| csc |                                |        |
